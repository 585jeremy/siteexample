# SGCNR Website

This folder is ready to upload to GitHub Pages as a static website.

## Quick publish steps

1. Create a new public repository on GitHub.
2. Upload every file from this folder into the root of that repository.
3. Open the repository on GitHub.
4. Go to **Settings** -> **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and the `/ (root)` folder.
7. Save and wait for GitHub Pages to publish the site.

Your site will usually appear at:

`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

Because this website uses hash routes like `#/rules` and `#/map`, it works well on GitHub Pages without extra routing setup.
